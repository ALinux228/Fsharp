# Задание 2. 

## Рекурсия

Создать список из n одинаковых элементов (элемент вводится)

### Алгоритм решения

Рекурсивная функция делится на 2 случая.

Базовый случай: если n = 0, то функция возвращает 1, при умножении на 1 результат не меняется

Рекурсивный случай: 

1. Берется последняя цифра числа

2. Функция вызывает сама себя для числа без последней цифры

3. Результаты перемножаются

# Задание 3. 

Создайте собственные функции для выполнения основных операций над списками (добавление/удаление/поиск элемента, сцепка двух списков, получение элемента по номеру).

## Задача 1. 
### Добавление

Реализовать функцию вставки элемента в список на указанную позицию.

### Алгоритм решения

1. Если список пустой и позиция равна 0, создаем новый список из элемента

2. Если список пустой и позиция не 0 - ошибка

3. Если позиция равна 0, вставляем элемент перед головой списка

4. Иначе сохраняем голову и рекурсивно вставляем элемент в хвост с позицией меньше на единицу.

## Задача 2. 

### Удаление

Реализовать функцию удаления элемента из списка на указанной позиции.

### Алгоритм решения

1. Если список пуст - ошибка

2. Если позиция равна 0, возвращаем хвост списка (удаляем первый элемент)

3. Иначе сохраняем голову и рекурсивно удаляем элемент из хвоста с позицией, меньшей на единицу

## Задача 3. 

### Поиск элемента по значению

Реализовать функцию поиска первой позиции элемента в списке по его значению.

### Алгоритм решения

1. Если список пуст - элемент не найден

2. Если голова списка равна искомому значению, возвращаем 0

3. Иначе рекурсивно ищем в хвосте и прибавляем 1 к найденной позиции

### Поиск элемента по номеру

Реализовать функцию получения элемента по номеру.

### Алгоритм решения

1. Если список пуст - ошибка

2. Если позиция равна 0, возвращаем голову списка

3. Иначе рекурсивно ищем элемент в хвосте с позицией, меньшей на единицу

## Задача 5. 

### Сцепка двух списков

Реализовать функцию сцепки двух списков

### Алгоритм решения

1. Если первый список пуст, возвращаем второй список

2. Иначе сохраняем голову первого списка и рекурсивно сцепляем хвост первого списка со вторым списком
